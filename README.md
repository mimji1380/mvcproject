# MVC Blog Project

This is a simple blog project built with Node.js, Express, and SQLite.

I created this project to practice MVC structure, REST API, and basic backend development.  
This project is mainly for university learning and practice.

## Technologies
- Node.js
- Express.js
- SQLite
- JavaScript
- HTML / CSS

## Features
- Create new posts
- Edit posts
- Delete posts
- View all posts
- Store data in SQLite database
- Show created and updated time
- Automatic dark mode (based on system)
